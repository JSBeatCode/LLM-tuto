============================================================
  [Topic 5] 멀티 체인 연결 패턴
============================================================

============================================================
X ▶ Step 1. 기본 멀티 체인 연결
           chain1 출력 → chain2 입력으로 자동 연결
============================================================

[체인 구조]
  chain1 : prompt1 | llm | StrOutputParser
  chain2 : {country: chain1} | prompt2 | llm | StrOutputParser

[결과]
조지 워싱턴, 에이브러햄 링컨, 마틴 루터 킹 주니어

============================================================
X ▶ Step 2. RunnableParallel + .assign()
           중간 결과(country) + 최종 결과(people) 함께 출력
============================================================

[체인 구조]
  chain3 : RunnableParallel(country=chain1).assign(people=chain2)

[결과 타입]: <class 'dict'>

[결과]
  country : 미국
  people  : 조지 워싱턴, 에이브러햄 링컨, 마틴 루서 킹 주니어

============================================================
▶ Step 3. Lambda — invoke dict에서 외부 매개변수 추출
============================================================

[체인 구조]
  RunnableParallel(country=chain1, num=lambda x: x['num'])
  → prompt2 | llm | StrOutputParser

[결과]
조지 워싱턴, 마틴 루서 킹 주니어, 앤디 워홀

============================================================
▶ Step 4. Lambda + .assign() — 중간 과정 전체 보존
============================================================

[체인 구조]
  RunnableParallel(country=chain1, num=lambda x: x['num'])
  .assign(res=chain4)

[결과 타입]: <class 'dict'>

[결과]
  country : 대한민국
  num     : 3
  res     : 이순신, 세종대왕, 김구

============================================================
▶ Step 5. JsonOutputParser + 체인 간 데이터 전달 응용
           JSON key → 다음 프롬프트 변수에 자동 매핑
============================================================

[체인 구조]
  chain1 : prompt1 | llm | JsonOutputParser  → {'actor':..., 'movie':...}
  chain2 : chain1 | prompt2 | llm | StrOutputParser
           ↑ JSON key가 {actor}, {movie} 변수에 자동 매핑

[결과]
리오나르도 디카프리오는 영화 "Inception"에서 주인공인 도미닉 "돔" 코브(Dominic "Dom" Cobb) 역할을 맡았습니다. 코브는 꿈을 통해 사람의 잠재의식에 침투하여 정보를 훔치는 전문 도둑입니다. 영화는 그가 복잡한 꿈의 구조 속에서 임무를 수행하고, 자신의 과거와 싸우는 과정을 그립니다.
