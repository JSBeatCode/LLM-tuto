[코드 설명]
1. 먼저 이 코드의 등장인물부터

이 코드에는 크게 4명이 있다고 보면 돼.

1) get_news(query)
실제로 네이버 뉴스 API를 호출하는 함수
검색어를 받아서 뉴스 결과 문자열을 만들어 반환

즉:

“검색은 내가 할게” 담당

2) Web_Search
LLM에게 보여주는 도구 설명서
“이 도구는 query라는 값을 받아서 검색하는 도구야”
검색어는 최대 2단어, 핵심 명사만 쓰라고 규칙 제공

즉:

“LLM아, 이런 형식으로 함수 호출해” 설명서

3) news_bot_v2(messages, ...)
OpenAI API를 호출하는 함수
LLM에게 메시지를 보내고,
필요하면 tool call도 하게 만드는 함수

즉:

“LLM에게 물어보는 창구”

4) news_qa_v2(prompt, ...)
전체 진행을 총괄하는 메인 함수
사용자 질문을 받아서
먼저 LLM에게 “필요한 검색어 뽑아봐” 시키고
LLM이 만든 여러 tool call을 실제로 실행하고
그 결과를 다시 LLM에게 주어 최종 답변을 생성

즉:

“전체 오케스트라 지휘자”

[코드 전체 요약]
사용자질문 = "넷플릭스 왓챠 디즈니플러스 웨이브 신작 추천해줘"

# 1차 LLM 호출
tool_calls = LLM에게_질문하고_필요한검색목록받기(사용자질문)

tool_results = []

# 각 검색 실행
for call in tool_calls:
    result = 네이버뉴스검색(call.query)
    tool_results.append(result)

# 2차 LLM 호출
최종답변 = LLM에게_검색결과전달하고_답변생성하기(
    사용자질문,
    tool_calls,
    tool_results
)

print(최종답변)
