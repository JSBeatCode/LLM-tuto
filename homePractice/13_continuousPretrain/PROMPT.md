이 코드를 공부할거야. 
이 코드를 공부할때에 주제로 나누어서 각 주제별로 하나의 '.py'파일로 만들어서 실행하는게 좋은지, 아니면 나누지 않고 하나의 '.py'로 만들어서 실행하는게 좋은 코드인지 먼저 판단해줘. 일단 나누지 말지 판단만 해줘.

---
[코딩 참고사항]
1.이 주제로 실행 가능하도록 하나의 '.py' 파일을 만들고 싶어.

3.모든 API_KEY나 API 관련 데이터를 불러오는 방식은 '.env' 파일을 생성하고 'from dotenv import load_dotenv'라는 코드를 활용해줘.

7.API 호출 시 너무 과다한 호출은 자제해 줘.
